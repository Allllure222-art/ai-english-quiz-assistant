在浏览器中完成（仅首次或换账号时需要）：

1）打开下面这一行链接（整行复制到地址栏）：

https://vercel.com/new?repository-url=https%3A%2F%2Fgithub.com%2FAllllure222-art%2Fai-english-quiz-assistant

2）按页面提示选择账号、Import 仓库，Framework 选 Next.js（一般自动识别）。

3）在 Vercel 项目 → Settings → Environment Variables 中至少配置：`OPENAI_API_KEY`、`NEXTAUTH_SECRET`、`NEXTAUTH_URL`（填 `https://ai-english-quiz-assistant.vercel.app`，无末尾斜杠）、若用 GitHub 登录则还有 `GITHUB_ID` / `GITHUB_SECRET`。保存后 Redeploy。详见 README「部署到公网」。
